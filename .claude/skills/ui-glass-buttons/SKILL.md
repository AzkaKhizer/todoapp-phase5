# Skill: Glassmorphism Buttons

## Purpose
Create modern glassmorphism-style buttons suitable for premium UI designs.

## Design Style
- Frosted glass look
- Semi-transparent background
- Soft blur backdrop
- Elegant borders
- Subtle glow on hover

## Core Visual Rules
- Background: rgba(255, 255, 255, 0.15)
- Backdrop blur: 10–20px
- Border: 1px solid rgba(255, 255, 255, 0.25)
- Border radius: 12–16px
- Box shadow: soft and layered

## Interaction States
### Hover
- Slight lift: translateY(-1px)
- Increased opacity
- Gentle glow

### Active
- Scale down: 0.97
- Reduce shadow

### Focus
- Visible ring
- Keyboard accessible

## Variants
- Primary (glass + accent glow)
- Secondary (neutral glass)
- Outline glass
- Icon button
- Disabled (no blur, muted opacity)

## Accessibility
- Minimum touch target: 44px
- Keyboard navigable
- High contrast text
- aria-label when icon-only

## Output Expectations
When creating buttons:
- Prefer reusable components
- Match project tech stack
- Do not overuse gradients
- Keep animations subtle

## Do Not
- Do not use harsh shadows
- Do not remove blur effect
- Do not reduce accessibility
