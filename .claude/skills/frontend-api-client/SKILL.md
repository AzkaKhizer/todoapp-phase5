# Frontend API Client Skill

## When to Use This Skill

- Calling backend APIs
- Attaching auth headers
- Handling API errors

## Procedure

1. **Read API base URL**
2. **Attach Authorization header**
3. **Handle HTTP errors**
4. **Return typed responses**

## Output Format

- Fetch wrapper
- Auth header injection
- Error handling logic

## Quality Criteria

- Centralized API logic
- Token-safe handling
- Clean error messages

## Example

**Input**: "Call protected backend API"

**Output**:
- Authenticated fetch helper
