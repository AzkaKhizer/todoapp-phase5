# Skill: Modern Navbar

## Purpose
Create a clean, modern, responsive navigation bar suitable for professional websites and web apps.

## Design Style
- Minimal and clean
- Modern typography
- Good spacing
- Optional glassmorphism support

## Layout Rules
- Logo on the left
- Navigation links centered or right-aligned
- CTA button optional on the right
- Sticky on top (optional)

## Desktop Behavior
- Horizontal menu
- Hover underline or subtle highlight
- Clear active link state

## Mobile Behavior
- Hamburger menu
- Slide-down or slide-in menu
- Large tap-friendly links
- Close menu on link click

## Animations
- Smooth open/close
- Subtle hover effects
- No heavy or flashy animations

## Accessibility
- Keyboard navigable
- aria-label for menu button
- Focus-visible styles
- Proper nav semantics

## Output Expectations
When creating a navbar:
- Do not break layout
- Keep responsive behavior intact
- Match existing design system
- Use reusable components

## Do Not
- Do not overcrowd navbar
- Do not use large animations
- Do not hide focus indicators
