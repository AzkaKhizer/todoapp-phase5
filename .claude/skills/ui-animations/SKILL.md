# Skill: UI Animations & Motion

## Purpose
Add subtle, elegant animations that enhance UX without distraction.

## Animation Philosophy
- Smooth
- Minimal
- Purpose-driven
- Never flashy

## Default Timing
- Duration: 150–300ms
- Easing: ease-out / cubic-bezier

## Common Animations
- Fade in
- Slide up (4–8px)
- Scale (0.98 → 1)
- Hover lift

## Button Motion
- Hover: lift + glow
- Active: scale down
- Disabled: no animation

## Card Motion
- Hover elevation
- Optional background glow

## Page Motion
- Staggered entry
- Respect reduced-motion preferences

## Tools
- CSS transitions
- Framer Motion (if React/Next.js)

## Accessibility
- Respect prefers-reduced-motion
- No infinite animations

## Do Not
- No bounce effects
- No long delays
- No heavy parallax by default
